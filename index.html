wp-simple-instagram

Wordpress Simple Instagram Plugins

Shortcode : [sig count="5" size="thumbnail" class="ig" heading="Your Heading"]

Parameter Description
count number of media to show
class add custom class before the gallery
size size of the image, there only 3 available options
thumbnail = 150x150
low_resolution = 320x320
standard_resolution = 640x640
heading Heading of the gallery (not mandatory)
